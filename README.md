# Showcase Portfolio

This repository contains my personal showcase portfolio. I created this website to present who I am as a designer and developer, while also showing my skills in front-end development, interaction, and visual design.

---

## 🎯 Purpose

The goal of this project is to create a portfolio that does more than just show my work. I want it to reflect my personality, interests, and the type of digital experiences I like to create.

Instead of making a standard portfolio, I chose to experiment with interactive effects, animations, and a strong hero section to make it stand out.

This portfolio is also part of my semester work and is used as evidence for my learning outcomes.

---

## ✨ Features

- Interactive hero section  
- Mouse-based animations (shaders)  
- Custom styling and typography  
- Subtle animations for better user experience  
- Project sections to present my work  

---

## 🧠 Process

I started by researching other portfolio websites and collecting inspiration for layouts, animations, and styling.

After that, I experimented with different concepts like mouse shaders and interactive elements. I also created prototypes and tested parts of the design with users to check if everything was clear and easy to use.

The project is still evolving as I continue to improve both the design and development.

---

## 🛠️ Tech Stack

- JavaScript
